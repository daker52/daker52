### 🚀 O mně
Jsem softwarový architekt a vývojář se zaměřením na **škálovatelné systémy** a **automatizaci**. Skrze svou značku **K&KCode** dodávám řešení, která kombinují kreativitu frontendu s neprůstřelnou logikou backendu.

- 🏗️ **Core:** Specializuji se na **Next.js**, **PHP (Laravel/Custom)** a orchestraci v **AWS**.
- 🧠 **AI Integration:** Implementuji LLM moduly a AI agenty pro zefektivnění procesů.
- ⚙️ **Systems:** Mám zkušenosti s **C++** pro výkonově náročné úlohy a herní logiku.
- 🌐 **Web:** Stavím na pevných základech **HTML5**, **CSS3** a **moderním JavaScriptu**.
